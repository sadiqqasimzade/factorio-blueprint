# Factorio Blueprint Generator

A comprehensive web application that converts images, videos, and audio into Factorio blueprints. Transform your creativity into in-game structures with our easy-to-use conversion tools.

- **Image to Blueprint Converter**: Convert PNG, JPG, GIF, WEBP, and SVG images into Factorio tile or lamp blueprints
- **Video to Blueprint Converter**: Transform videos into animated lamp displays with frame-by-frame conversion
- **Pixel Art Tools**: Create custom pixel art designs optimized for Factorio
- **Space Platform Converter**: Generate space platform layouts from transparent images
- **Blueprint Decoder/Encoder**: Decode and encode blueprint strings for sharing and editing


If you don't have Node installed, you can use the provided Dockerfile:

```bash

# Run the container
docker run -p 3000:3000 factorio-blueprint-generator
```
